# Claude Project System

A reusable methodology and template for managing large projects with Claude Code.

## Key Files

| File | Purpose |
|------|---------|
| `METHOD.md` | Full methodology documentation (human-readable) |
| `template/` | Starter files to copy when creating a new project |
| `template/setup.sh` | One-command project initialization |
| `template/generate.js` | Dashboard generator |
| `template/build-module.sh` | Sequential ticket builder |
| `template/modules.conf` | Module-to-feature-dir mapping |
| `template/CLAUDE.md` | Agent instructions template |
| `template/docs/` | Template docs (PRD, kanban, module READMEs) |
